インストール

インストールは

  • DLL とアドインファイル（xla）を所定の場所にコピー
  • Excel 上でアドインとして登録

の手順で行います．Windows Vista より前と以降，Excel 2007 より前と以降でアドインファイルの保存場所，アドインの登録方法が異なるので，それぞれ以下の方法でインストールを行ってください．

DLL とアドインファイル（xla）のコピー

Windows 2000, Windows XP

  • ZIPファイルをダウンロード，解凍後に以下の場所に DLL, アドインファイル（xla）をコピーします（解凍したファイル群に srats2010.dll と srats2010.xla があることを確認して下さい）．
  • アドインフォルダ（\Documents and Settings\（ユーザ名）\Application Data\Microsoft\AddIns）に srats2010.dll と srats2010.xla をコピーします．

Windows Vista, Windows 7

  • ZIPファイルをダウンロード，解凍後に以下の場所に DLL, アドインファイル（xla）をコピーします（解凍したファイル群に srats2010.dll と srats2010.xla があることを確認して下さい）．
  • アドインフォルダ（\Users\（ユーザ名）\AppData\Roaming\Microsoft\AddIns）に srats2010.dll と srats2010.xla をコピーします．

アドインの登録

Excel 2002, Excel XP, Excel 2003

  • Excel を起動します．
  • Excel のメニューバーから ツール -> アドイン... を選択します．
  • SRATS にチェックを入れます．
  • 上記の操作でメニューバーに SRATS の項目が現れればインストール完了です．

Excel 2007, Excel 2010

  • Excel を起動します．
  • Excel のメニュー上部のオフィスマークから Excel のオプションを選択します．
  • アドインパネルの「設定...」を選択します．
  • Srats2010 にチェックを入れます．
  • 上記の操作でメニュータブのアドインに SRATS の項目が現れればインストール完了です．

「無効なオブジェクトライブラリです。または定義されていないオブジェクトへの参照を含んでいます。」のエラーが出る場合は下記を試してください．

  • コマンドプロンプトを管理者として実行する。
    • アクセサリ
    • コマンドプロンプトを右クリック
    • 管理者として実行
  • 次のコマンドを入力して実行：regsvr32 C:\Windows\System32\MSCOMCTL.OCX
  • 64bit OS の場合は次のコマンドにしてください：regsvr32 C:\Windows\SysWOW64\MSCOMCTL.OCX

アンインストール

  • Excel を起動します．
  • アドイン登録の操作と同様にして Srats2010 のチェックを外します．
  • コピーした DLL ファイルと XLA ファイルを消去します．
