動作環境

ハードウェア構成

ソフトウェア構成に示すOSが動作するパーソナルコンピュータが必要です．

ソフトウェア構成

  1. OS
    以下のOSでの動作が確認されています．
    • Microsoft Windows 2000 Professional
    • Microsoft Windows XP Home Edition
    • Microsoft Windows XP Professional Edition
  2. アプリケーション
    以下のアプリケーションでの動作が確認されています．
    • Microsoft Excel 2000
    • Microsoft Excel 2002
    • Microsoft Excel 2003

インストール

ZIPファイルをダウンロード後に以下の手順でインストールします．
  1. ZIP ファイルを適当な場所で解凍します．
    （解凍したファイル群に SRMDLL.dll と SRATS.xla あることを確認して下さい）
  2. アドインフォルダ（\Documents and Settings\（ユーザ名）\Application Data\Microsoft\AddIns）に SRMDLL.dll と SRATS.xla をコピーします．
    （コピー後は解凍したファイル群を消去してもかまいません）
  3. Excel を起動します．
  4. Excel のメニューバーから ツール -> アドイン... を選択します．
  5. SRATS にチェックを入れます．
上記の操作でメニューバーに SRATS の項目が現れればインストール完了です．

アンインストール

  1. Excel を起動します．
  2. Excel のメニューバーから ツール -> アドイン... を選択します．
  3. SRATS のチェックを外します．
  4. インストールの手順２でコピーした DLL ファイルと XLA ファイルを消去します．
